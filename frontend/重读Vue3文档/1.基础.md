#### 创建一个Vue应用

​	每个Vue应用都是通过 ==createApp==  函数创建一个新的 **应用实例**。

```vue
import { createApp } from 'vue'
// 从一个spa组件中导入根组件
// 每个应用都需要一个 根组件，其他组件将作为其 子组件。(本质就是一个（组件）对象，根组件是Vue渲染的起点)

import App from './App.vue'

const app = createApp(App)
app.mount('#app')
```
